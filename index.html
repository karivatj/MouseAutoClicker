Mouse Autoclicker

Mouseclick Automation Tool by Kari Vatjus-Anttila made with Python 2.7, PyQT4 and pals.

A tool to automate arbitrary mouseclick sequences.

Download the release here!

Instructions

Either download the release executable from the releases page or download the source package and run AutoClicker with a python intepreter. Any suggestions or feature requests can be forwarded to kari.vatjusanttila [{at}] gmail.com
